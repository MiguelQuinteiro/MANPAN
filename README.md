# MANPAN 🖥️🛠️

Este repositorio constituye el motor de **Mantenimiento de Pantallas** e interfaces de usuario. Es el módulo encargado de estandarizar la capa de presentación, permitiendo una gestión dinámica y coherente de los formularios en todo el ecosistema de software.

## 🚀 Propósito del Proyecto

Desarrollar una infraestructura que facilite la creación, actualización y mantenimiento de interfaces gráficas. **MANPAN** permite que los sistemas de **B & M Sistemas IA** presenten una imagen unificada, optimizando la interacción del usuario y reduciendo el tiempo de desarrollo de nuevos módulos de entrada de datos.

## 📁 Estructura del Repositorio

Organizado para una gestión eficiente de la capa visual:

* **src/**: 
  * **/Templates**: Formularios base (`.frm`) con estructuras predefinidas para ABM (Altas, Bajas y Modificaciones).
  * **/Controls**: Configuraciones personalizadas para controles ActiveX y componentes visuales estándar.
  * **/Themes**: Módulos para la gestión de colores, fuentes y estilos visuales dinámicos.
* **docs/**: Guías de diseño de interfaz, estándares de usabilidad y mapas de navegación de pantallas.
* **lib/**: Librerías de iconos y recursos gráficos utilizados en las interfaces.

## 🛠️ Capacidades Técnicas

* **Estandarización UI/UX**: Garantiza que todos los formularios sigan una lógica de navegación y diseño idéntica, facilitando la curva de aprendizaje del usuario.
* **Mantenibilidad**: Centraliza los cambios visuales; una modificación en el template de **MANPAN** se refleja en todo el sistema.
* **Integración con DB**: Vinculación optimizada entre los controles de pantalla y las estructuras de datos de los proyectos `DBF` y `SQL`.
